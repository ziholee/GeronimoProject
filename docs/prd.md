# 🧠 Discord Bot 기능 추가 PRD
## 파티 맺기 예약 시스템 (Party Scheduling System)

---

## 1. 📌 문서 목적
본 문서는 기존 Discord 봇에 “파티 모집 및 자동 집합 시스템” 기능을 추가하기 위한 요구사항을 정의한다.  
해당 기능은 사용자들이 특정 시간에 함께 모일 수 있도록 자동화된 파티 모집 및 음성채널 초대 기능을 제공하는 것을 목표로 한다.

본 기능은 **discord.js v14 기반**으로 구현한다.

---

## 2. 🎯 기능 개요

### 2.1 한 줄 설명
슬래시 명령어로 파티를 생성하면 모집 메시지가 생성되고, 유저가 이모지로 참여 의사를 표시하며, 예약 시간이 되면 봇이 음성채널을 생성하고 참가자에게 DM으로 초대 링크를 전송한다.

### 2.2 기대 효과
- 파티 모집 자동화
- 참여 인원 관리 간소화
- 집합 시간 준수율 증가
- 운영자 개입 최소화
- 커뮤니티 활성화

---

## 3. 👤 사용자 정의

### 3.1 파티 생성자
- 파티를 생성하는 유저
- 시간 및 조건 설정
- 결과 확인

### 3.2 참가자
- 이모지 반응으로 참여
- DM으로 초대 링크 수신

### 3.3 관리자
- 봇 권한 관리
- 문제 발생 시 대응

---

## 4. 🧩 사용자 시나리오

### 4.1 파티 생성
1. 유저가 `/파티생성` 입력
2. 파티 정보 입력 (제목, 시간 등)
3. 봇이 모집용 임베드 메시지 생성
4. 메시지에 참여용 이모지 표시

---

### 4.2 파티 참여
1. 유저가 이모지 클릭
2. 참여자로 등록됨
3. 중복 참여는 제한

---

### 4.3 파티 시작
1. 예약 시간 도달
2. 봇이 음성채널 생성
3. 초대 링크 생성
4. 참가자에게 DM 발송
5. 파티 생성자에게 결과 전달

---

## 5. ⚙️ 기능 요구사항

### 5.1 명령어
- `/파티생성`

#### 입력 옵션
- 제목 (필수)
- 설명 (선택)
- 집합 시간 (필수)
- 모집 마감 시간 (선택)
- 최대 인원 (선택)
- 채널 이름 (선택)

---

### 5.2 모집 메시지 (Embed)
- 파티 제목
- 설명
- 생성자
- 집합 시간
- 모집 마감
- 참여 방법 안내
- 현재 참여 인원
- 상태 (모집중 / 시작됨)

---

### 5.3 참여 방식
- 이모지 반응 (예: ✅)
- 중복 참여 불가
- 모집 마감 후 참여 불가
- 최대 인원 초과 시 제한

---

### 5.4 음성채널 생성
- 예약 시간 도달 시 자동 생성
- 채널 이름 규칙 적용
- 임시 채널로 운영

---

### 5.5 초대 링크 생성
- 생성된 음성채널 기반
- 참가자에게 DM으로 전송

---

### 5.6 DM 알림
내용 포함:
- 파티 제목
- 집합 시간
- 음성채널 링크
